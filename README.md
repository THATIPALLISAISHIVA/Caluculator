# Calculator Project

A simple, responsive web-based calculator application built using vanilla HTML, CSS, and JavaScript.

## Features

- **Basic Arithmetic**: Perform addition, subtraction, and multiplication.
- **Clear Function**: Reset calculations instantly with the "AC" (All Clear) button.
- **Live Evaluation**: Calculate results dynamically using standard order of operations.
- **Responsive Design**: Centered layout with hover effects and a clean user interface.
- **Error Handling**: Alerts the user for invalid expressions.

## Technologies Used

- **HTML5**: For the structure of the application.
- **CSS3**: For styling, including grid layouts, flexbox, and transitions.
- **JavaScript**: For the calculator logic and DOM manipulation.

## How to Run

1. **Clone the Repository** (or download the files):
   ```bash
   git clone <repository_url>
   ```
2. **Open the Application**:
   - Navigate to the project folder.
   - Double-click on `caluculator.html` to open it in your default web browser.

## Functionality

- **Numbers**: Click 0-9 to enter numbers.
- **Operators**: Use `+`, `-`, `*` for arithmetic operations.
- **Decimal**: Use `.` for decimal numbers.
- **Result**: Press `=` to see the calculation result.
- **Reset**: Press `AC` to clear the screen.

## Project Structure

- `caluculator.html`: The main HTML file containing the structure and styles.
- `caluculator.js`: The JavaScript file handling the application logic.
- `README.md`: Project documentation.
